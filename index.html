Introduction

CSS is a language developers use to style HTML. CSS is critical to be familar with for front-end development. Even if you are focused on back-end development, it is helpful to understand some basic CSS in order to be able to efficiently read and understand code you may be expected to work with or build upon in the future. It is also a lot of fun to use and to learn, as you can see its impact immediately!

Keep reading to learn more!

Why Should I Learn CSS?

As you build on your coding knowledge and skills over time, you will find that there are libraries and frameworks that you can learn that help with styling, like Bootstrap. First, though, you want to learn basic CSS so that you can build your coding knowledge on top of a solid foundation and understand how elements are impacted by code. After all, if you learned how to do a layup before practicing ball handling, you wouldn't be all that useful in a game time scenario.

What Do I Already Need to Know?

Before you can learn basic CSS, you need to learn basic HTML. Once you have HTMl written, you use CSS to make the .html document look good. If you don't have a foundation in HTML yet, check out at least one of the following resources before moving ahead:

Setting Up a CSS file

Since you already know that you need an HTML file to begin styling, now you can create and link a .css file to your .html file.

You have already stored your .html file in a folder that can serve as a repo. Now you will create a new folder inside titled css. Create a blank file titled style.css inside the css folder.

Now you will link your .css file to your html file. Open your .html file in your console of choice. Within the head section, insert the following code:

<link rel="stylesheet" href="css/style.css">

Save the file. Now your css stylesheet is linked to your html file and ready to code!

Coding CSS

With CSS, every element or object in your html code can be targeted and stylized. The syntax of CSS is as follows:

HTML element {
  property1: value;
  property2: value;
}

For example, you can target the font in your h2 as follows.

h2 {
  font-family: 'Arial';
  font-size: 20px;
  color: Orchid;
  text-decoration: overline;
}

This is more efficient than using HTML tag for h2 to target these properties as it will apply to all the h2 tags on the page. The same stylesheet can also be linked to multiple pages in your folder/repo, making consistent styling a breeze!

Classes

If you assign a class to elements in your HTML code, you can group elements to style them in CSS. For example, perhaps you want some of your h2 elements to be the color aquamarine, font size 24px, and bold and some of your h2 elements to be the color orchid, font size 30px, and italicized. You can give each h2 element in your HTML a class:

<h2 class="orchid">
<h2 class="aquamarine">

Now in the style.css file, you can style by the given class (CSS shorthand for class is . ):

.orchid {
  color: orchid;
  font-size: 30px;
  font-style: italic;
}

.aquamarine {
color: aquamarine;
font-size: 24px;
font-weight: bold;
}
